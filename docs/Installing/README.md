# How to install

1. Download via git:
    - `git clone https://github.com/Tsunami014/Blaze-Sudio/`
    - Go to [the website](https://github.com/Tsunami014/Blaze-Sudio/) and download the repo as a zip file then unzip it

2.  ```bash
    pip install -r requirements.txt
    ```
